مستودع حزم ADR (Arch Developers Repositories)

هذا المستودع مخصص لاستضافة حزم Pacman الخاصة بمشروع ADR.

لإضافة المستودع إلى نظام Arch Linux/Trch Linux الخاص بك، أضف السطر التالي إلى ملف pacman.conf:

[adr-stable]
SigLevel = Optional TrustAll
Server = https://adembenarfa175-code.github.io/adr-repo/$arch
